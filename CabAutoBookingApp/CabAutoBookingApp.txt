Q: What is the use of Tkinter in Python?
A: Tkinter is used to create graphical user interfaces (GUIs) for desktop applications.

Q: What is a Combobox used for?
A: A Combobox allows users to choose from a dropdown list of options.

Q: How can we display popup messages in Tkinter?
A: By using the messagebox module, such as messagebox.showinfo() or messagebox.showerror().

Q: How can we close the Tkinter window using code?
A: By calling root.destroy().

Q: What is the difference between Entry and Combobox widgets?
A: Entry allows users to type free text; Combobox allows selection from predefined options.

Q: How do you ensure all fields are filled before booking?
A: By checking if not all([...]) before proceeding.

Q: What module is used to get the current date and time?
A: The datetime module.

Q: Can we store the booking details in a file or database?
A: Yes, we can use Python file handling or sqlite3/MySQL database connection for permanent storage.
