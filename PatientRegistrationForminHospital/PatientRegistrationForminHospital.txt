Q: What is the purpose of Tkinter in Python?
A: Tkinter is a standard GUI library used to create desktop-based user interfaces.

Q: What is the difference between Entry and Text widgets?
A: Entry is used for single-line input, while Text allows multi-line input.

Q: What is the use of a Checkbutton?
A: It allows the user to select multiple options independently.

Q: What is the purpose of a Listbox?
A: It displays a list of items from which the user can select one or more options.

Q: What is messagebox used for?
A: It shows pop-up messages for feedback, errors, or confirmations.

Q: How do you read which item is selected in a Listbox?
A: Using listbox.curselection() and then listbox.get(index).

Q: What does .get("1.0", END) mean for the Text widget?
A: It retrieves all text from line 1, character 0 up to the end of the text box.
